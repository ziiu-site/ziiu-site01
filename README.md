## Quick Start

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## Customization

Edit content in:

- `src/config.ts` - Site configuration
- `src/content/*.ts` - Features, testimonials, FAQ
- `src/styles/globals.css` - Colors & animations
